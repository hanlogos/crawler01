## 🎮 완전한 기능 목록

### 크롤링 제어
- ✅ ▶ 시작
- ✅ ⏸ 일시정지

- ✅ 파일명 자동 생성 (원본/AI 선택)

- ✅ 검색 기능
- ✅ 컨텍스트 메뉴
- ✅ 상세 정보 패널
- ✅ 일괄 AI 분석
- ✅ 일괄 저장

### 전략 시스템

## 🚀 사용 방법

### 기본 실행
```bash
python enhanced_crawling_dashboard.py
```

### 주요 기능
1. **크롤링 운영 탭**
   - 사이트별 상태 모니터링
   - 전체 제어 버튼
   - 개별 제어 버튼
   - 보고서 목록 및 관리

2. **시간대별 전략 탭**
   - 시간대별 최적 설정 확인
   - 전략 적용

3. **리스크 관리 탭**
   - 현재 리스크 레벨 확인
   - 복구 프로토콜 확인

4. **AI 인사이트 탭**
   - 운영/데이터 관리/데이터 활용 조언
