### 주요 경고 유형 상세

#### 1. 타입 힌트 부족 (540개, 91%)
- **유형**: `missing_type_hint`, `missing_return_type`
- **의미**: 함수 파라미터 또는 반환값에 타입 힌트가 없음
- **예시**:
  ```python
  # 경고 발생
  def analyze(self, stock_code):  # 타입 힌트 없음
      return result
  
  # 개선 후
  def analyze(self, stock_code: str) -> Dict[str, Any]:
      return result
  ```

#### 2. Docstring 부족
- **유형**: `missing_docstring`, `missing_args_section`
- **의미**: 함수에 Docstring이 없거나 Args/Returns 섹션이 없음
- **예시**:
  ```python
  # 경고 발생
  def calculate(self, value):
      return value * 2
  
  # 개선 후
  def calculate(self, value: float) -> float:
      """
      값을 2배로 계산
      
      Args:
          value: 입력 값
      
      Returns:
          계산된 값 (2배)
      """
      return value * 2
  ```
